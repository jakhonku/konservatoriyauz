# O'zbekiston Davlat Konservatoriyasi Veb-sayti

Ushbu loyiha O'zbekiston Davlat Konservatoriyasi uchun zamonaviy, akademik va to'liq boshqariladigan veb-sayt loyihasidir.

## Xususiyatlari
- **Asosiy sahifa**: "Ken Burns" effekti, premium tipografiya va responsiv dizayn.
- **Admin Panel (`/admin`)**: Matbuot kotibi uchun yangiliklarni boshqarish tizimi.
- **Media**: Rasm va videolarni yuklash va ularni "glassmorphism" uslubida ko'rsatish imkoniyati.
- **Glassmorphism**: Navigatsiya paneli shaffof va xira oyna stilida.
- **Backend API**: Express orqali ma'lumotlarni saqlash va boshqarish.

## Ishga tushirish
1. `run_all.bat` faylini ishlating.
2. Brauzerda `http://localhost:5173` manziliga kiring.

## Texnologiyalar
- React + Vite
- Tailwind CSS 4
- Express + Multer
- Lucide React (Ikonkalar)
